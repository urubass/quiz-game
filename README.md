## Requirements

- Node.js >=18

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the server:
   ```bash
   node server.js
   ```
   or use:
   ```bash
   npm start
   ```

The server will be available on `http://localhost:3000` by default.
